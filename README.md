# linux-learning-notes

This repository contains notes and commands I learned while setting up and managing a Linux server using Ubuntu.

## Topics Practiced

- Navigating the Linux file system
- Managing files and directories
- File permissions
- User management
- Installing packages
- SSH remote access
- Basic system administration

## Useful Commands

### Navigation
```bash
pwd
ls
cd
```

## File Management
```bash
mkdir test
touch file.txt
cp file.txt backup.txt
mv file.txt folder/
rm file.txt
```

## Permissions
```bash
chmod 755 file.sh
chown user:user file.txt
Package Management
sudo apt update
sudo apt upgrade
sudo apt install nginx
SSH Access
ssh username@server-ip
```

## Environment
Ubuntu Linux
Bash terminal
